# 도면2 문제 분석

## 사용자가 지적한 문제
1. **일람표 위에 마킹**: 좌측 하단 MEMBER LIST 영역에 빨간 원들이 표시됨
2. **SG2 수직선이 대각선에 마킹**: 일부 SG2가 수직선인데 대각선에 마킹되고 있음

## 현재 상황
- SG2 텍스트 위치: (432340.5, 106385.9)
- 이 위치는 일람표 영역 내부임
- 일람표 헤더 'MARK' @ (432293.1, 107723.6)
- 일람표 내용: SG2, H-250x125x6.0x9.0, SS275

## 문제 해석
이미지를 보면:
- 좌측 하단에 "MEMBER LIST" 표가 있음
- 그 위(도면 영역)에 대각선 구조물들이 있고, 그 위에 빨간 원들이 마킹되어 있음
- 빨간 원들이 일람표 **위**에도 표시되어 있다는 것은, 일람표 영역에 잘못 마킹되었다는 의미

## 실제 문제
1. 일람표 영역 감지가 부정확하여, 도면 영역의 일부가 일람표로 잘못 인식됨
2. 또는 일람표 위의 도면 영역에 있는 부호들이 일람표로 필터링되어 마킹되지 않음

## 해결 방향
일람표 영역을 더 정확하게 제한해야 함. 
현재 Y 범위가 너무 넓어서 도면 영역까지 포함하고 있을 가능성이 높음.
